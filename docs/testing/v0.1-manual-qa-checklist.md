# v0.1 manual QA checklist

Record environment: OS, browser/version, Node.js, viewport/device, audio output, reduced-motion setting, date, tester.

For every item record **PASS / FAIL / NOT TESTED / NOT APPLICABLE**, notes, and evidence.

## Participant

- 一献、色選択、pointer gesture、描き直し、代替gesture、約10秒の踊り
- 仲添、留添候補、自由入力、入力エラー、開栓Canvas
- 音声あり／なし、音量、停止、replay、reset、会場送信、duplicate
- 320/375/390/430pxで横スクロール、fixed action、focus、Canvasを確認

## Venue

- 0件、live受信、1件、複数件、新着表示、土地集計、reload、後開き、複数venue tab
- clearのキャンセル／確定、reduced motion、1280x720、1920x1080、console error

## Accessibility and recovery

- キーボード完走、focus表示、見出し・label、色以外の選択状態、Canvas代替情報
- 音声拒否／なし、壊れたlocalStorage、tab切替、開栓中reset、ネットワーク不要
